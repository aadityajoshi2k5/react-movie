# 🎬 MovieFinder

A simple movie search app built with **React**, **Vite**, and the **TMDB API**.

Users can search movies, view details, watch trailers, and explore trending movies.

---

## ✨ Features

- Search movies in real time
- View popular & trending movies
- Movie details page
- Watch trailers
- Responsive design
- Loading & error states
- Trending searches with Appwrite

---

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- TMDB API
- Appwrite

---

## 📦 Installation

### 1. Clone the project

```bash
git clone https://github.com/sarfarazstark/react-movie.git
cd react-movie
```

### 2. Install dependencies

```bash
npm install
```

### 3. Create `.env` file

```env
VITE_MOVIE_API_KEY=your_tmdb_api_key
VITE_APPWRITE_ENDPOINT=your_appwrite_endpoint
VITE_APPWRITE_PROJECT_ID=your_project_id
VITE_APPWRITE_DATABASE_ID=your_database_id
VITE_APPWRITE_COLLECTION_ID=your_collection_id
```

### 4. Start development server

```bash
npm run dev
```
